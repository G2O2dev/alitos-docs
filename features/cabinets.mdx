Раздел **Кабинеты** — центральная точка работы в Alitos. Здесь собраны все рекламные кабинеты, привязанные к вашему аккаунту: вы видите их статусы, балансы, привязки к менеджерам и другие ключевые параметры. Из этого же раздела можно создать новый кабинет ГЦК и скопировать нужные поля для вставки во внешние таблицы — без лишних переходов и ручного ввода.

## Что отображается в списке кабинетов

Список кабинетов показывает сводную информацию по каждому подключённому кабинету:

- **Название** кабинета и его идентификатор
- **Статус** — активен, приостановлен или заблокирован
- **Баланс** — текущий остаток средств
- **Рекламный менеджер (АМ)** — ответственный за кабинет
- **Принадлежность к ГЦК / мГЦК** — офис, к которому привязан кабинет
- **Дата последнего обновления** данных

Колонки можно сортировать, а строки — фильтровать с помощью сохранённых наборов фильтров. Подробнее о фильтрах читайте в разделе [Фильтры](/features/filters).

## Создание кабинета ГЦК из Alitos

Вы можете создать новый кабинет ГЦК прямо из интерфейса Alitos, не переходя во внешние системы. Для этого у вашего аккаунта должны быть соответствующие права.

<Steps>
  <Step title="Откройте раздел «Кабинеты»">
    В левом боковом меню выберите пункт **Кабинеты**. Загрузится список всех доступных вам кабинетов.
  </Step>
  <Step title="Нажмите «Создать кабинет»">
    В правом верхнем углу страницы нажмите кнопку **Создать кабинет**. Откроется форма создания нового кабинета ГЦК.
  </Step>
  <Step title="Заполните параметры кабинета">
    Укажите необходимые данные:
    - **Название кабинета** — понятное внутреннее название
    - **Офис ГЦК** — выберите из списка доступных офисов
    - **Рекламный менеджер** — назначьте ответственного АМ
    - Прочие обязательные поля формы
  </Step>
  <Step title="Подтвердите создание">
    Нажмите **Сохранить**. Новый кабинет появится в списке. Если данные успешно переданы в систему, статус кабинета сразу отобразится как активный.
  </Step>
</Steps>

## Копирование полей кабинета

Alitos позволяет быстро копировать значения полей кабинетов — это особенно удобно при заполнении внешних таблиц, отчётов или передаче данных коллегам.

### Копирование поля одного кабинета

Наведите курсор на нужное поле в строке кабинета (например, на ID или название). Появится иконка копирования — нажмите на неё. Значение будет скопировано в буфер обмена, и вы увидите кратковременное подтверждение на экране.

### Копирование полей нескольких кабинетов

Чтобы скопировать одно и то же поле сразу по нескольким кабинетам:

1. Отметьте нужные кабинеты с помощью чекбоксов в левой части списка.
2. В появившейся панели действий выберите **Копировать поле**.
3. Укажите, какое именно поле нужно скопировать (например, ID кабинета или название).
4. Все значения будут скопированы в буфер обмена в виде списка — по одному значению на строку.

<Tip>
  При копировании нескольких кабинетов значения разделяются переносом строки, поэтому их можно вставить сразу в столбец таблицы — в Google Sheets или Excel каждое значение попадёт в отдельную ячейку. Это удобно для массового составления отчётов.
</Tip>

## Советы по работе со списком

- **Используйте фильтры**, чтобы быстро находить нужные кабинеты по менеджеру, офису или статусу. Сохранённые наборы фильтров позволяют переключаться между видами в один клик.
- **Группируйте по АМ или мГЦК**, если работаете с большим количеством кабинетов — структурированный вид упрощает навигацию. Подробнее в разделе [Группировка](/features/grouping).
- **Сортируйте по балансу**, чтобы оперативно видеть кабинеты с низким остатком средств и своевременно их пополнять.
- **Обновляйте данные** вручную кнопкой обновления, если нужны актуальные значения прямо сейчас, не дожидаясь автоматической синхронизации.
