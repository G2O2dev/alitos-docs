Когда кабинетов становится много, навигация по списку без фильтрации превращается в рутину. Alitos решает эту проблему с помощью **именованных наборов фильтров** — вы один раз настраиваете нужные условия отбора, сохраняете их под понятным именем и впоследствии применяете в один клик. Дополнительно к фильтрам можно включить группировку по рекламному менеджеру (АМ) или по микро-ГЦК (мГЦК), чтобы видеть структурированный список вместо плоского перечня кабинетов.

## Что такое набор фильтров

Набор фильтров — это сохранённая конфигурация условий отбора кабинетов. Например, вы можете создать набор «Активные кабинеты АМ Иванова», который показывает только те кабинеты, где статус — «активен», а ответственный менеджер — конкретный АМ. В следующий раз, когда понадобится этот вид, достаточно выбрать набор из списка — повторно выставлять условия не нужно.

Наборы фильтров:
- **Сохраняются между сессиями** — доступны при каждом входе в систему
- **Применяются мгновенно** — список кабинетов обновляется сразу после выбора набора
- **Не удаляют данные** — фильтр только меняет отображение, все кабинеты остаются в системе
- **Могут совмещаться с группировкой** — применяйте фильтр и одновременно группируйте результат по АМ или мГЦК

## Создание набора фильтров

<Steps>
  <Step title="Откройте раздел «Кабинеты»">
    В левом меню перейдите в **Кабинеты**. Вы увидите полный список кабинетов без активных фильтров (или с ранее применённым набором).
  </Step>
  <Step title="Откройте панель фильтров">
    Нажмите кнопку **Фильтры** в верхней части страницы. Раскроется панель с доступными условиями отбора: статус, АМ, офис ГЦК / мГЦК, баланс и другие параметры.
  </Step>
  <Step title="Настройте условия отбора">
    Установите нужные значения для одного или нескольких параметров. Список кабинетов обновляется в реальном времени по мере того, как вы выбираете условия — так вы сразу видите результат.
  </Step>
  <Step title="Сохраните набор фильтров">
    Когда конфигурация готова, нажмите **Сохранить набор**. Введите понятное название — например, «Кабинеты мГЦК Север» или «Проблемные кабинеты». Нажмите **Сохранить**.
  </Step>
  <Step title="Проверьте сохранённый набор">
    Откройте выпадающий список наборов фильтров — новый набор должен появиться там. Выберите его, чтобы убедиться, что применяются именно те условия, которые вы настроили.
  </Step>
</Steps>

## Переключение между наборами фильтров

Все сохранённые наборы доступны в выпадающем меню рядом с кнопкой **Фильтры**. Чтобы применить нужный набор, просто выберите его из списка — фильтрация сработает немедленно. Чтобы сбросить все фильтры и вернуться к полному списку кабинетов, выберите пункт **Без фильтра** или нажмите **Сбросить**.

Ненужный набор можно удалить: откройте список наборов, наведите курсор на нужный пункт и нажмите иконку удаления рядом с его названием.

## Группировка кабинетов

Помимо фильтрации, Alitos поддерживает **группировку** — структурирование списка кабинетов по заданному признаку. Группировка и фильтрация работают независимо, но отлично сочетаются: сначала отфильтруйте нужные кабинеты, затем сгруппируйте их для удобного просмотра.

<Tabs>
  <Tab title="Группировка по АМ">
    **АМ** — рекламный менеджер, ответственный за группу кабинетов. Группировка по АМ разбивает список на секции, где каждая секция содержит кабинеты одного менеджера.

    **Когда это полезно:**
    - Вы руководитель и хотите видеть нагрузку по каждому менеджеру
    - Нужно быстро оценить объём кабинетов у конкретного АМ
    - Вы сам АМ и хотите видеть только «свою» секцию без лишних кабинетов

    **Как включить группировку по АМ:**
    1. В разделе **Кабинеты** нажмите **Группировка** (кнопка рядом с фильтрами).
    2. Выберите пункт **По рекламному менеджеру (АМ)**.
    3. Список кабинетов перестроится: каждая группа будет озаглавлена именем АМ, а под заголовком — все кабинеты этого менеджера.
    4. Группы можно сворачивать и разворачивать нажатием на заголовок.
  </Tab>
  <Tab title="Группировка по мГЦК">
    **мГЦК** (микро-ГЦК) — подразделение или офис внутри ГЦК, к которому привязана группа кабинетов. Группировка по мГЦК объединяет кабинеты по принадлежности к конкретному микро-офису.

    **Когда это полезно:**
    - Вы работаете с несколькими мГЦК и хотите видеть кабинеты каждого отдельно
    - Нужно быстро оценить состояние кабинетов в конкретном микро-ГЦК
    - Вы анализируете эффективность разных офисов и сравниваете их показатели

    **Как включить группировку по мГЦК:**
    1. В разделе **Кабинеты** нажмите **Группировка**.
    2. Выберите пункт **По мГЦК**.
    3. Список кабинетов перестроится по принадлежности к микро-ГЦК: каждая группа озаглавлена названием мГЦК.
    4. Кабинеты, не привязанные ни к одному мГЦК, отображаются в отдельной группе **Без мГЦК** в конце списка.
  </Tab>
</Tabs>
